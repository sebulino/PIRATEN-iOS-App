## Codebase Patterns
- Xcode project uses SwiftUI for UI
- Project structure follows Clean Architecture with App/, Core/, Features/, Resources/, Config/, Tests/, Docs/
- Use `xcodebuild -scheme PIRATEN -destination 'platform=iOS Simulator,name=iPhone 16'` for builds
- Files are organized: App entry point in App/, domain entities in Core/, assets in Resources/
- **Xcode 16 auto-discovery**: Project uses objectVersion 77 with PBXFileSystemSynchronizedRootGroup - files are automatically discovered from filesystem, no manual project.pbxproj editing needed
- Auth state management: Use @StateObject in App struct, @ObservedObject in child views
